# GraceTracks Icons

Place your PWA icon files here:

- `icon-192.png` — 192×192 px
- `icon-512.png` — 512×512 px

Both should be square, with transparent or solid background.
The `maskable` purpose requires safe-zone padding (at least 10% on all sides).

Recommended tool: https://maskable.app for testing maskable icons.
